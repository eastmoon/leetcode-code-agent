# Code Summarization

程式語言 : C++

## 摘要

於輸入字串中尋找不含重複字元之連續子字串，並求其最大長度。子字串是否含重複字元之判定，逐一檢視子字串內各字元；若某字元已於同一子字串中出現過，判定為含重複字元，否則判定為不含重複字元。針對輸入字串所有可能之連續子字串，依起始位置與結束位置窮舉，逐一判定是否不含重複字元，並於判定成立時比較其長度與目前已知最大長度，取較大值更新之。程式提供兩個邏輯相同之最大長度計算程序，分別供主流程之不同呼叫使用。主流程依序對三組字串執行計算，並輸出各次計算所得之最大長度。

## 範例

**Example 1:**

```
Input: s = "abcabcbb"
Output: 3
```

**Example 2:**

```
Input: s = "bbbbb"
Output: 1
```

**Example 3:**

```
Input: s = "pwwkew"
Output: 3
```

## 約束

給定字串 S，長度為 n。

子字串範圍以起始索引 i 與結束索引 j 表示，其中 0 <= i <= j <= n-1。

子字串 S[i..j] 不含重複字元之條件：不存在 p, q，其中 i <= p < q <= j，使得 S[p] = S[q]。

輸出 = max(j - i + 1)，其中 S[i..j] 不含重複字元，窮舉範圍為 i 由 0 遞增至 n-1，對每一 i，j 由 i 遞增至 n-1。

n = 0 時，輸出為 0。
