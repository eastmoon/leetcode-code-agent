# Code Summarization

程式語言 : C++

## 摘要

由兩個以鏈結串列表示之非負整數相加，計算其加總結果，並以鏈結串列形式回傳；串列各節點依十進位由低位至高位排列。相加過程逐位對齊兩串列對應節點之數值，並加上前一位計算所得之進位值；若該位總和達十以上，記錄進位並取其扣除十之餘值為該位數值，否則不記錄進位並直接取總和為該位數值；任一串列已達尾端時，該串列對應位之數值視為零；兩串列皆達尾端後，若仍有進位，於結果串列尾端新增一節點記錄該進位。程式另提供由數值清單建立鏈結串列，以及依序輸出鏈結串列各節點數值（節點間以分隔符號區隔）之輔助程序。主流程分別建立兩組鏈結串列並執行相加，再依序輸出各次相加所得之結果串列數值。

## 範例

**Example 1:**

```
Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]
```

**Example 2:**

```
Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Output: [8,9,9,9,0,0,0,1]
```

**Example 3:**

```
Input: l1 = [0], l2 = [0]
Output: [0]
```

## 約束

給定鏈結串列 L1，長度 m；給定鏈結串列 L2，長度 n。

節點依十進位由低位至高位排列，每一節點數值 ∈ [0, 9]。

結果串列長度 = max(m, n)，末位計算後進位值為 1 時，結果串列長度 = max(m, n) + 1。

第 k 位計算：sum(k) = L1[k]（k < m，否則為 0）+ L2[k]（k < n，否則為 0）+ carry(k-1)。

若 sum(k) >= 10，該位數值 = sum(k) - 10，carry(k) = 1；否則該位數值 = sum(k)，carry(k) = 0。

carry(-1) = 0。
