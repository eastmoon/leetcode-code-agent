# Code Summarization

程式語言 : C++

## 摘要

由兩個陣列之元素合併為單一集合，排序此合併集合，再依集合元素數量之奇偶，計算並回傳其中位數。元素數量為奇數時，中位數為排序後之中間位置元素；元素數量為偶數時，中位數為排序後中間兩個位置元素之平均值。程式提供兩個邏輯相同之中位數計算程序，分別供主流程之不同呼叫使用。主流程分別建立兩組陣列並執行計算，再依序輸出各次計算所得之中位數，數值格式化至小數點後五位。

## 範例

**Example 1:**

```
Input: nums1 = [1,3], nums2 = [2]
Output: 2.00000
```

**Example 2:**

```
Input: nums1 = [1,2], nums2 = [3,4]
Output: 2.50000
```

**Example 3:**

```
Input: nums1 = [], nums2 = [1]
Output: 1.00000
```

## 約束

給定陣列 A，長度 m；給定陣列 B，長度 n。

合併集合 C，為 A 與 B 之元素聯集（保留重複值），長度 k = m + n，經排序使 C[0] <= C[1] <= ... <= C[k-1]。

若 k mod 2 = 1，中位數 = C[k/2]（整數除法，索引由 0 起算）。

若 k mod 2 = 0，中位數 = (C[k/2 - 1] + C[k/2]) / 2。
