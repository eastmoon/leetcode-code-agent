# Code Summarization

程式語言 : C++

## 摘要

從數字清單中尋找兩個索引位置相異之數字，使兩數字相加之結果等於指定目標值，並將此兩數字所在之索引位置依序記錄至結果清單。比對依索引由小至大依序進行，取首次成立之組合。比對成立後以全域旗標標記已找到，並中止後續比對。程式提供兩個邏輯相同之比對程序，分別供主流程之不同呼叫使用。主流程依序對三組數字清單與目標值執行比對，並於每次呼叫前重置全域旗標，再輸出各次比對所得之索引位置。

## 範例

**Example 1:**

```
Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: nums[0] + nums[1] = 2 + 7 = 9
```

**Example 2:**

```
Input: nums = [3,2,4], target = 6
Output: [1,2]
Explanation: nums[1] + nums[2] = 2 + 4 = 6
```

**Example 3:**

```
Input: nums = [3,3], target = 6
Output: [0,1]
Explanation: nums[0] + nums[1] = 3 + 3 = 6
```

## 約束

給定數字清單 A，長度為 n；給定目標值 t。

存在 i, j，其中 i, j ∈ [0, n-1] 且 i ≠ j，使得 A[i] + A[j] = t。

搜尋順序：i 由 0 遞增至 n-1；對每一 i，j 由 0 遞增至 n-1。

輸出為首次滿足條件之 (i, j)，依序為結果清單第一、第二元素。

結果清單元素數量 = 2（條件成立時）；未成立時結果清單元素數量 = 0。

全域旗標之值 ∈ {0, 1}；呼叫前之值須由外部設定，條件成立時設為 1。
